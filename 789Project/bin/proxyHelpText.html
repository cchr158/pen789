Proxy Settings Help

This help section will help you complete the ZAP proxy settings for pen789. You should not modifie the proxy settings unless you are fermilar with the ZAP API.

  1. Proxy Address:

    The Local ZAP proxy address is the address where the ZAP proxy can be found. If you have manually configured a ZAP proxy then enter its address here.

  2. Proxy Port:

    The Local ZAP proxy port is the port that the ZAP proxy is listening on.

  3. ZAP API Key:

    The ZAP API Key is used to prevent other applications from accessing the ZAP API without permission. If you have more that one ZAP session on the go you can switch between them by changing the text in this feild to their respective API keys.

More information is avalable at https://www.owasp.org/index.php/OWASP_Zed_Attack_Proxy_Project.
